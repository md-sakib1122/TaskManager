Question-1: What are the different ways to select an element in the DOM?

getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML: Returns/sets the HTML content as a string, 2.innerText: Returns/sets visible text, ignoring hidden elements, 3.textContent: Returns/sets all text, including hidden elements.

Question-3: What is event delegation in the DOM?

A technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

An event starts from the target element and propagates up through its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

1.Create: document.createElement('tag'), 2.Add: parent.appendChild(child) or parent.insertBefore(child, reference) 3.Remove: element.remove() or parent.removeChild(child)
